# Face Detection with YOLO + Gradio
Проект по обнаружению лиц на изображениях с использованием архитектуры YOLO и интерактивного веб-интерфейса на основе Gradio.
Модель обучена самостоятельно на датасете “Humans” с Kaggle и сохранена в формате PyTorch.
> Простой запуск локального веб-приложения
>⚠️ Проект исследовательский — не предназначен для продакшена без доработки
## Особенности
- Обнаружение лиц на статичных изображениях
- Предобученная модель: best_model_phash.pt
- Веб-интерфейс через Gradio (запускается локально)
- Поддержка загрузки изображений через браузер
- Автоматическая визуализация bounding box’ов
- Обучено на реалистичном датасете Humans (Kaggle) — разные ракурсы, освещение, масштабы
## Как запустить
1. Установите зависимости
```bash
pip install -r requirements.txt
```
2. Запустите приложение:
```bash
python app.py
```
3.  Откройте в браузере ссылку, указанную в терминале (обычно http://127.0.0.1:7860)
4.  Загрузите изображение — модель автоматически выделит все найденные лица.

## О датасете
- Название: Humans (доступен на Kaggle)
- Содержание: Изображения людей в естественных условиях — разное освещение, позы, расстояния до камеры
- Цель: Обучение модели надёжно находить лица независимо от масштаба и ориентации
